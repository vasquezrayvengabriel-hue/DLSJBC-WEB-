# De La Salle John Bosco College Website

## Overview

This is a responsive and professional website for De La Salle John Bosco College. The website provides information about the school, admissions, news, and contact details.

---

## Features

- Responsive Design
- Professional User Interface
- Sticky Navigation Bar
- Quick Navigation Menu
- School Profile Header
- Image Gallery
- Admission Information
- News & Announcements
- Contact Page
- Google Map
- GitHub Pages Ready

---

## Pages

- Home
- About
- Admission
- Posts
- Contact Us
